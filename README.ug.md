# 🐱 شياۋشياۋ يائو ئېلىكتېرونلۇق ئەرمىكى

PySide6 ئاساسىدا قۇرۇلغان، چىرايلىق Windows ئېلىكتېرونلۇق ئەرمەك پروگراممىسى. «شياۋشياۋ يائو» ئۈستىلىڭىزدا مېڭىدۇ، يۈگۈرىدۇ، ئۇسسۇل ئوينايدۇ ۋە سىز بىلەن پاراڭلىشىدۇ！

## ✨ ئىقتىدار ئالاھىدىلىكلىرى

- **جانلىق ئانىماتسىيە** — ئالتە خىل ھەرىكەت：ساقلىشىش، مېڭىش، يۈگۈرۈش، ئۇسسۇل ئويناش، سالاملىشىش، بىزار بۇلۇش
- **AI پاراڭلىشىش** — DeepSeek API ئارقىلىق، شياۋشياۋ يائو ئۆزىگە خاس مىجەز بىلەن سىزگە جاۋاب بېرىدۇ
- **سىستېما ساندۇقى** — سىستېما ساندۇقىغا كىچىكلەپ، خىزمىتىڭىزگە دەخلى قىلمايدۇ
- **سۆرەش** — شياۋشياۋ يائونى ئۈستەلنىڭ خالايدىغان ئورنىغا سۆرەپ قويغىلى بولىدۇ
- **ئۆز-ئارا تەسىرلىشىش** — قوش چېكىش بىلەن ئىنكاس قايتۇرىدۇ، كۆپ چېكىپ قالسىڭىز ئاچچىقلىنىپ قېچىپ كېتىدۇ 😂
- **بۇلۇت سىنكرونلاشتۇرۇش** — Supabase ئارقىلىق بۇلۇت سىنكرونلاشتۇرۇشنى قوللايدۇ（ئىختىيارى）
- **بىلىم ئامبىرى** — ئۈندىدار سۆزلىشىش خاتىرىسىنى سەپلەپ، شياۋشياۋ يائوغا سۆزلىشىش ئۇسلۇبىڭىزنى ئۆگەتكىلى بولىدۇ
- **ئاپتوماتىك قوزغىلىش** — كومپيۇتېر قوزغالغاندا ئاپتوماتىك قوزغىلىشنى قوللايدۇ
- **كۆپ تىل** — خەنزۇچە ۋە ئۇيغۇرچە قوش تىللىق سالاملاشنى قوللايدۇ

## 🛠 تېخنىكا

- **Python 3.9+**
- **PySide6** — Qt for Python ئۈستەل رامكىسى
- **DeepSeek API** — AI سۆزلىشىش موتورى
- **Supabase** — بۇلۇت سانلىق مەلۇمات سىنكرونلاشتۇرۇش（ئىختىيارى）
- **Pillow** — رەسىم بىر تەرەپ قىلىش
- **jieba** — خەنزۇچە تىل بۆلۈش

## 📦 قاچىلاش ۋە ئىجرا قىلىش

### 1. كلونلاش

```bash
git clone https://github.com/你的用户名/شياۋشياۋ يائو ئېلىكتېرونلۇق ئەرمىكى.git
cd شياۋشياۋ يائو ئېلىكتېرونلۇق ئەرمىكى/pet_app
```

### 2. كېرەكلىك بۇيرۇقلارنى قاچىلاش

```bash
pip install -r requirements.txt
```

### 3. API Key تەڭشەش

```bash
copy config.example.json config.json
```

```json
{
  "api_key": "sk-DeepSeek-API-Key",
  "model_name": "deepseek-v4-flash",
  "always_on_top": true,
  "auto_start": false,
  "pet_size": 300
}
```

> 🔑 DeepSeek API Key ئېلىش：[platform.deepseek.com](https://platform.deepseek.com)

### 4. ئىجرا قىلىش

```bash
python main.py
```
